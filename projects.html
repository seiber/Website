Pong Game

This project is a (3 month 4 person project) multiplayer pong based java game with client-server based architecture, it has player movement, menu options, a login system with username and password storage. I set up the GUIS for client-server connection, and the networking with java ChatServers. Users would pass in the port number, and could connect to another user who was hosting the game from different computers to play together. At the end of the project, we had a team presentation, where we each explained our setup, and our contributions.
